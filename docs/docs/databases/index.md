# Databases

This section covers the various databases the `sunholo` library helps interact with to support your GenAI application VAC.

## Supported Databases

- [AlloyDB](./alloydb.md) - Google AlloyDB with pgvector
- [PostgreSQL](./postgres.md) - Standard PostgreSQL with vector extensions
- [LanceDB](./lancedb.md) - Embedded vector database
- [Supabase](./supabase.md) - Supabase integration
- [Discovery Engine](./discovery_engine.md) - Google AI Search
- [Firestore](./firestore.md) - Google Cloud Firestore with circuit breaker and retry
